# e-plantShopping

A React-based e-commerce web application for browsing and purchasing plants online. Built as part of an IBM/Coursera React capstone project.

## Overview

e-plantShopping lets users explore a curated catalog of plants organized into categories, view pricing, and place orders through a simple shopping cart flow.

## Features

- **Browse Plants** – View a wide selection of plants available for purchase
- **Categorized Listings** – Plants are organized into categories for easy discovery:
  - Air-purifying plants
  - Aromatic/fragrant plants
  - Insect-repellent plants
  - Medicinal plants
- **Pricing Display** – Each plant listing shows its price clearly
- **Add to Cart** – Add any plant to your shopping cart
- **Cart Management** – Review items, quantities, and totals before checkout
- **Place Order** – Complete the purchase flow directly through the app

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS

## Getting Started

### Prerequisites

- Node.js installed on your machine
- npm (comes with Node.js)

### Installation

1. Clone the repository
   ```bash
   git clone <your-repo-url>
   ```
2. Navigate into the project folder
   ```bash
   cd e-plantShopping
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm run dev
   ```
5. Open the local URL shown in your terminal (usually `http://localhost:5173`) to view the app
